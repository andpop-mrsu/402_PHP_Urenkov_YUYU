# Calculator Game

Консольная арифметическая игра "Калькулятор" на PHP.

## Описание игры

Игроку показывается случайное арифметическое выражение с операциями `+`, `-`, `*`, содержащее четыре операнда (например `35+16*2-4`), которое нужно вычислить и записать ответ. После этого программа выводит соответствующее сообщение и правильный ответ.

## Установка

### Глобальная установка (рекомендуется):

**text**

```
composer global require werrys3021/calculator
```

После установки игра будет доступна из любой директории командой `calculator`.

### Локальная установка:

**text**

```
composer require werrys3021/calculator
```

## Использование

**text**

```
calculator
```

Пример сессии игры:

**text**

```
Добро пожаловать в игру 'Калькулятор'!
Введите ваше имя: Yura
Выражение: 12+5*3-8
Введите ответ: 19
Правильно!
```

## Разработка

1. Клонируйте репозиторий:

**text**

```
git clone https://github.com/werrys3021/calculator.git
cd calculator
```

2. Установите зависимости:

**text**

```
composer install
```

3. Запустите игру в режиме разработки:

**text**

```
php bin/calculator
```

* PHP 7.4 или выше
